## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
ArtifactResponse DTO, ArtifactListResponse DTO. GET /api/v1/artifacts (with query params: q, type, harness, tag, page, size). GET /api/v1/artifacts/{slug}/{version}. GET /api/v1/artifacts/{slug}/latest. GET /api/v1/artifacts/{slug}/members — for agent-group type, returns ordered list of member AgentGroupMember DTOs (slug, role, resolved metadata).

- [ ] #2 GET /api/v1/artifacts/{slug}/{version} returns full artifact content

## Implementation Plan

<!-- SECTION:PLAN:BEGIN -->
1. Create package com.agentlibrary.web.api

2. Create ArtifactResponse DTO (src/main/java/com/agentlibrary/web/api/ArtifactResponse.java)
   - Java record with fields:
     * String name (slug)
     * String title
     * String type (slug form)
     * String version
     * String description
     * List<String> harnesses (slug form)
     * List<String> tags
     * String category, language, author, visibility
     * String created, updated (ISO-8601 strings)
     * String content (body text)
   - Static factory: fromArtifact(Artifact artifact) → ArtifactResponse
   - Static factory: fromMetadata(ArtifactMetadata meta) → ArtifactResponse (content=null)

3. Create ArtifactListResponse DTO (src/main/java/com/agentlibrary/web/api/ArtifactListResponse.java)
   - Java record with fields:
     * List<ArtifactResponse> items
     * int page
     * int size
     * long totalElements
     * int totalPages
   - Static factory: of(List<ArtifactMetadata> items, int page, int size, long total)

4. Create GroupMemberResponse DTO (src/main/java/com/agentlibrary/web/api/GroupMemberResponse.java)
   - Java record: String slug, String role, ArtifactResponse metadata
   - Maps from ResolvedGroupMember

5. Create ArtifactApiController (src/main/java/com/agentlibrary/web/api/ArtifactApiController.java)
   - @RestController @RequestMapping("/api/v1/artifacts")
   - Inject ArtifactService

6. Implement GET /api/v1/artifacts (AC #1, AC #4)
   - Method: list(@RequestParam(required=false) String q,
                  @RequestParam(required=false) String type,
                  @RequestParam(required=false) String harness,
                  @RequestParam(required=false) String tag,
                  @RequestParam(defaultValue="0") int page,
                  @RequestParam(defaultValue="20") int size)
   - Build Filter from params (parse type→ArtifactType, harness→Harness, or null if absent)
   - If q is present: call artifactService.search(q, filter) for full-text search
   - If q is absent: call artifactService.list(filter) for index listing
   - Apply pagination (page, size) to results
   - Return ArtifactListResponse with paginated subset

7. Implement GET /api/v1/artifacts/{slug}/{version} (AC #2, AC #3)
   - Method: get(@PathVariable String slug, @PathVariable String version)
   - Call artifactService.get(slug, version)
   - NotFoundException propagates to ApiExceptionHandler → 404 (AC #3)
   - Map Artifact → ArtifactResponse, return 200

8. Implement GET /api/v1/artifacts/{slug}/latest
   - Method: getLatest(@PathVariable String slug)
   - Call artifactService.get(slug, "latest")
   - NotFoundException → 404 (AC #3)
   - Return ArtifactResponse

9. Implement GET /api/v1/artifacts/{slug}/members (AC #5, AC #6)
   - Method: getMembers(@PathVariable String slug)
   - Call artifactService.resolveGroup(slug)
   - If artifact is not AGENT_GROUP type, throw IllegalArgumentException → mapped to 400 (AC #6)
   - Map List<ResolvedGroupMember> → List<GroupMemberResponse>
   - Return 200 with ordered member list (AC #5)

10. Handle path conflict between /{slug}/members and /{slug}/{version}:
    - Spring MVC routes literal "members" before path variable {version}
    - No regex constraint needed — literal segments have higher priority
    - Verify with test that GET /artifacts/my-skill/members routes to getMembers()

11. Unit tests (src/test/java/com/agentlibrary/web/api/ArtifactApiControllerTest.java)
    - @WebMvcTest(ArtifactApiController.class)
    - @MockBean ArtifactService
    - @WithMockUser for authentication
    - Test GET /api/v1/artifacts returns 200 with paginated JSON (AC #1)
    - Test GET /api/v1/artifacts?q=git&type=skill filters correctly (AC #4)
    - Test GET /api/v1/artifacts?harness=claude&tag=workflow filters (AC #4)
    - Test GET /api/v1/artifacts/git-helper/1.0.0 returns full artifact (AC #2)
    - Test GET /api/v1/artifacts/unknown-slug/1.0.0 returns 404 (AC #3)
    - Test GET /api/v1/artifacts/git-helper/latest returns latest version
    - Test GET /api/v1/artifacts/my-group/members returns ordered list (AC #5)
    - Test GET /api/v1/artifacts/my-skill/members returns 400 for non-group (AC #6)
    - All tests use MockMvc + Jackson assertions on JSON response body

12. Dependencies: Requires TASK-0005.01 (ArtifactService), TASK-0005.02 (SecurityConfig for auth context)
