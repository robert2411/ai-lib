## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
SemVer class parsing MAJOR.MINOR.PATCH(-prerelease) strings. Comparable implementation. Static isValid() and parse() methods. Rejects invalid strings with descriptive exception.
<!-- SECTION:DESCRIPTION:END -->

## Implementation Plan

<!-- SECTION:PLAN:BEGIN -->
1. Independent task — no dependencies on other Milestone 2 tasks
2. Create class SemVer in com.agentlibrary.model.SemVer:
   - File: src/main/java/com/agentlibrary/model/SemVer.java
   - Implements: Comparable<SemVer>
   - Final class (not a record — needs custom compareTo and parsing logic)
   - Private fields: int major, int minor, int patch, String prerelease (nullable)
   - Private constructor: SemVer(int major, int minor, int patch, String prerelease)
   - Public getters: major(), minor(), patch(), prerelease()
   - Static method: public static SemVer parse(String version)
     - Regex pattern: ^(0|[1-9]\d*)\.(0|[1-9]\d*)\.(0|[1-9]\d*)(?:-([-a-zA-Z0-9]+(?:\.[-a-zA-Z0-9]+)*))?$
     - Throws IllegalArgumentException with descriptive message if format invalid
     - Extracts major, minor, patch as ints; prerelease as nullable String
   - Static method: public static boolean isValid(String version)
     - Returns true if parse would succeed, false otherwise (try/catch internally)
   - Override: public int compareTo(SemVer other)
     - Compare major, then minor, then patch (numeric ordering)
     - Pre-release comparison per semver 2.0.0 spec:
       - A version WITHOUT prerelease has HIGHER precedence than same version WITH prerelease
       - i.e. 1.0.0 > 1.0.0-alpha
       - Among pre-releases: split by ".", compare each identifier:
         - Numeric identifiers compared as integers
         - Alphanumeric identifiers compared lexically (case-sensitive)
         - Numeric < alphanumeric
         - Fewer fields < more fields (if all preceding are equal)
   - Override: equals(Object) and hashCode() — based on major, minor, patch, prerelease
   - Override: toString() — returns "MAJOR.MINOR.PATCH" or "MAJOR.MINOR.PATCH-prerelease"
3. Create test file SemVerTest:
   - File: src/test/java/com/agentlibrary/model/SemVerTest.java
   - Test: parseSimple — SemVer.parse("1.0.0") has major=1, minor=0, patch=0, prerelease=null
   - Test: parseWithPrerelease — SemVer.parse("1.0.0-alpha.1") has prerelease="alpha.1"
   - Test: ordering — 1.0.0 < 1.2.0 < 2.0.0 (AC #1)
   - Test: orderingMinor — 1.0.0 < 1.1.0 < 1.2.0
   - Test: orderingPatch — 1.0.0 < 1.0.1 < 1.0.2
   - Test: prereleaseOrdering (AC #2):
     - 1.0.0-alpha < 1.0.0-alpha.1 < 1.0.0-alpha.beta < 1.0.0-beta < 1.0.0-beta.2 < 1.0.0-beta.11 < 1.0.0-rc.1 < 1.0.0
   - Test: prereleaseHasLowerPrecedence — 1.0.0-alpha < 1.0.0
   - Test: invalidStrings (AC #3):
     - "" throws IllegalArgumentException
     - "not.a.version" throws
     - "1.0" throws (missing patch)
     - "1.0.0.0" throws (extra segment)
     - "01.0.0" throws (leading zero)
     - "1.0.0-" throws (trailing hyphen, empty prerelease)
   - Test: isValidTrue — isValid("1.2.3") returns true
   - Test: isValidFalse — isValid("garbage") returns false
   - Test: toStringSimple — SemVer.parse("2.1.0").toString() equals "2.1.0"
   - Test: toStringPrerelease — SemVer.parse("1.0.0-beta.1").toString() equals "1.0.0-beta.1"
   - Test: equalityAndHashCode — parse("1.0.0").equals(parse("1.0.0")), different versions not equal
   - Test: compareToEqual — parse("1.0.0").compareTo(parse("1.0.0")) == 0
