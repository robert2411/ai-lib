**`init()` implementation detail:**

```java
@PostConstruct
public void init() {
    Path repoDir = Path.of(properties.getRepoPath());
    if (Files.exists(repoDir.resolve("HEAD"))) {
        // Already initialised — open existing bare repo
        LOG.info("Opening existing bare repo at " + repoDir);
        this.repository = new FileRepositoryBuilder()
            .setGitDir(repoDir.toFile())
            .setBare()
            .build();
    } else {
        // First run — create bare repo
        LOG.info("Creating new bare repo at " + repoDir);
        Files.createDirectories(repoDir);
        this.repository = new FileRepositoryBuilder()
            .setGitDir(repoDir.toFile())
            .setBare()
            .build();
        this.repository.create(true); // true = bare
    }
}
```

Key JGit API usage:
- `FileRepositoryBuilder.setGitDir().setBare().build()` — opens or prepares a bare repo
- `repository.create(true)` — creates the bare repo on disk (objects/, refs/, HEAD, etc.)
- Check for `HEAD` file to detect existing repo (standard git indicator)

**`close()` implementation:**
```java
@PreDestroy
public void close() {
    if (repository != null) {
        repository.close();
    }
    writeQueue.shutdown();
}
```
