### Implementation Steps

#### Step 1: Implement `list(Filter filter)`

Reads INDEX.yaml from HEAD tree and applies filters.

```java
@Override
public List<ArtifactMetadata> list(Filter filter) {
    try {
        String yaml = readFileFromHead("INDEX.yaml");
        List<ArtifactMetadata> all = IndexFile.load(yaml);
        return all.stream()
            .filter(m -> matchesFilter(m, filter))
            .toList();
    } catch (IOException e) {
        throw new StorageException("Failed to list artifacts", e);
    }
}

private boolean matchesFilter(ArtifactMetadata meta, Filter filter) {
    if (filter == null) return true;
    if (filter.type() != null && meta.type() != filter.type()) return false;
    if (filter.harness() != null && !meta.harnesses().contains(filter.harness())) return false;
    if (filter.tag() != null && !meta.tags().contains(filter.tag())) return false;
    if (filter.query() != null && !filter.query().isBlank()) {
        String q = filter.query().toLowerCase();
        boolean matches = (meta.name() != null && meta.name().toLowerCase().contains(q))
            || (meta.title() != null && meta.title().toLowerCase().contains(q))
            || (meta.description() != null && meta.description().toLowerCase().contains(q));
        if (!matches) return false;
    }
    return true;
}
```

Notes:
- `list` is a read operation — does NOT go through WriteQueue
- Empty filter (all null fields) returns all artifacts (AC#1)
- Query filter does case-insensitive substring match on name, title, description
- Filter by visibility is not in the Filter record currently — not required by AC

#### Step 2: Implement `versions(String type, String name)`

Scan all tags matching pattern `<type>/<name>@*` and return sorted list.

```java
@Override
public List<VersionRef> versions(String type, String name) {
    try {
        String tagPrefix = type + "/" + name + "@";
        List<VersionRef> refs = new ArrayList<>();
        
        List<Ref> tagRefs = repository.getRefDatabase()
            .getRefsByPrefix(Constants.R_TAGS + tagPrefix);
        
        try (RevWalk rw = new RevWalk(repository)) {
            for (Ref ref : tagRefs) {
                String tagName = ref.getName().substring(Constants.R_TAGS.length());
                // tagName = "skill/my-skill@1.0.0", we want "1.0.0"
                String version = tagName.substring(tagName.lastIndexOf("@") + 1);
                
                ObjectId commitId = ref.getObjectId();
                RevCommit commit = rw.parseCommit(commitId);
                Instant timestamp = Instant.ofEpochSecond(commit.getCommitTime());
                
                refs.add(new VersionRef(version, commitId.name(), timestamp));
            }
        }
        
        // Sort by SemVer descending (AC#3)
        refs.sort((a, b) -> SemVer.parse(b.version()).compareTo(SemVer.parse(a.version())));
        return Collections.unmodifiableList(refs);
    } catch (IOException e) {
        throw new StorageException("Failed to list versions for " + type + "/" + name, e);
    }
}
```

Key JGit APIs:
- `repository.getRefDatabase().getRefsByPrefix(String)` — returns refs matching prefix
- `RevWalk.parseCommit()` — gets commit timestamp
- Tag format: `refs/tags/skill/my-skill@1.0.0`

#### Step 3: Implement `delete(String type, String name, String message, User user)`

Wrapped in WriteQueue. Removes the artifact folder from the tree and updates INDEX.yaml.

```java
@Override
public void delete(String type, String name, String message, User user) {
    writeQueue.submit(() -> {
        doDelete(type, name, message, user);
        return null;
    });
}

private void doDelete(String type, String name, String message, User user) throws IOException {
    ArtifactType artType = ArtifactType.fromSlug(type);
    String folderPath = artType.folderName() + "/" + name + "/";
    
    // 1. Verify artifact exists in current tree
    ObjectId headTree = getHeadTree();
    if (headTree == null) {
        throw new StorageException("Cannot delete from empty repository");
    }
    
    // 2. Build new tree WITHOUT the artifact folder (TreeWalk skip-and-add)
    DirCache dc = DirCache.newInCore();
    DirCacheBuilder builder = dc.builder();
    
    try (ObjectReader reader = repository.newObjectReader();
         TreeWalk tw = new TreeWalk(reader)) {
        tw.addTree(headTree);
        tw.setRecursive(true);
        
        boolean found = false;
        while (tw.next()) {
            String path = tw.getPathString();
            if (path.startsWith(folderPath)) {
                found = true;
                continue; // skip — this is what we are deleting
            }
            // Copy all other entries
            DirCacheEntry entry = new DirCacheEntry(path);
            entry.setObjectId(tw.getObjectId(0));
            entry.setFileMode(tw.getFileMode(0));
            builder.add(entry);
        }
        
        if (!found) {
            throw new StorageException("Artifact not found: " + type + "/" + name);
        }
    }
    
    builder.finish();
    
    try (ObjectInserter inserter = repository.newObjectInserter()) {
        ObjectId newTreeId = dc.writeTree(inserter);
        inserter.flush();
        
        // 3. Commit deletion
        String commitMsg = type + "/" + name + ": " + (message != null ? message : "delete");
        PersonIdent author = new PersonIdent(user.username(), user.username() + "@library");
        PersonIdent committer = new PersonIdent("agent-library", "server@agent-library");
        ObjectId commitId = createCommit(newTreeId, commitMsg, author, committer);
        updateHead(commitId);
        
        // 4. Update INDEX.yaml (remove entry)
        String currentYaml = readFileFromHead("INDEX.yaml");
        List<ArtifactMetadata> entries = new ArrayList<>(IndexFile.load(currentYaml));
        entries.removeIf(e -> e.type() == artType && e.name().equals(name));
        String updatedYaml = IndexFile.save(entries);
        
        byte[] indexBytes = updatedYaml.getBytes(StandardCharsets.UTF_8);
        ObjectId blobId = insertBlob(indexBytes);
        ObjectId indexTreeId = buildTree("INDEX.yaml", blobId);
        ObjectId indexCommitId = createCommit(indexTreeId, "chore: update INDEX.yaml", committer, committer);
        updateHead(indexCommitId);
    }
}
```

Note: Tags are NOT deleted (per design doc: "Delete: commit removes folder; tags remain").

#### Step 4: Implement `bundle(String type, String name, String ref)`

Creates a **tar.gz** archive of a single artifact at a specific version/ref, matching the
design doc and REST endpoint `GET /artifacts/{type}/{name}/bundle?ref=…`.

Dependency: Apache Commons Compress (`org.apache.commons:commons-compress`) for tar.gz support.
Alternatively, use `java.util.zip.GZIPOutputStream` + manual tar format, but Commons Compress
is cleaner and handles POSIX tar correctly.

```java
@Override
public InputStream bundle(String type, String name, String ref) {
    try {
        ArtifactType artType = ArtifactType.fromSlug(type);
        String folderPath = artType.folderName() + "/" + name + "/";
        
        // Resolve tree from ref (version tag) or HEAD
        ObjectId treeId;
        if (ref == null || ref.isBlank() || "latest".equals(ref)) {
            treeId = getHeadTree();
        } else {
            // Resolve version ref via tag
            String tagRef = Constants.R_TAGS + type + "/" + name + "@" + ref;
            Ref tagObject = repository.exactRef(tagRef);
            if (tagObject == null) {
                throw new StorageException("Version not found: " + type + "/" + name + "@" + ref);
            }
            try (RevWalk rw = new RevWalk(repository)) {
                RevCommit commit = rw.parseCommit(tagObject.getObjectId());
                treeId = commit.getTree().getId();
            }
        }
        
        if (treeId == null) {
            throw new StorageException("Cannot bundle from empty repository");
        }
        
        // Build tar.gz archive of all files in the artifact folder
        ByteArrayOutputStream baos = new ByteArrayOutputStream();
        try (GzipCompressorOutputStream gzOut = new GzipCompressorOutputStream(baos);
             TarArchiveOutputStream tarOut = new TarArchiveOutputStream(gzOut)) {
            tarOut.setLongFileMode(TarArchiveOutputStream.LONGFILE_POSIX);
            
            try (TreeWalk tw = new TreeWalk(repository)) {
                tw.addTree(treeId);
                tw.setRecursive(true);
                
                boolean found = false;
                while (tw.next()) {
                    String path = tw.getPathString();
                    if (!path.startsWith(folderPath)) continue;
                    found = true;
                    
                    // Strip the folder prefix for archive-relative paths
                    // e.g. "skills/git-helper/skill.md" -> "skill.md"
                    String archivePath = path.substring(folderPath.length());
                    
                    ObjectId blobId = tw.getObjectId(0);
                    byte[] content = repository.open(blobId).getBytes();
                    
                    TarArchiveEntry tarEntry = new TarArchiveEntry(archivePath);
                    tarEntry.setSize(content.length);
                    tarOut.putArchiveEntry(tarEntry);
                    tarOut.write(content);
                    tarOut.closeArchiveEntry();
                }
                
                if (!found) {
                    throw new StorageException("Artifact not found: " + type + "/" + name);
                }
            }
            
            tarOut.finish();
        }
        
        return new ByteArrayInputStream(baos.toByteArray());
    } catch (IOException e) {
        throw new StorageException("Failed to create bundle for " + type + "/" + name, e);
    }
}
```

Key design decisions:
- Format is tar.gz per design doc (not ZIP)
- Single artifact at a specific ref (not multi-slug)
- Throws StorageException if artifact or version not found (fail-fast)
- Archive paths are relative to the artifact folder (not full repo paths)
- Uses Commons Compress: `GzipCompressorOutputStream` + `TarArchiveOutputStream`

#### Step 5: Implement `get(String type, String name, String ref)`

Read a specific artifact (latest or at a version ref):

```java
@Override
public Optional<Artifact> get(String type, String name, String ref) {
    try {
        ArtifactType artType = ArtifactType.fromSlug(type);
        String primaryFile = getPrimaryFileName(artType);
        String path = artType.folderName() + "/" + name + "/" + primaryFile;
        
        ObjectId treeId;
        if (ref == null || ref.isBlank() || "latest".equals(ref)) {
            treeId = getHeadTree();
        } else {
            // Resolve version ref via tag
            String tagRef = Constants.R_TAGS + type + "/" + name + "@" + ref;
            Ref tagObject = repository.exactRef(tagRef);
            if (tagObject == null) return Optional.empty();
            try (RevWalk rw = new RevWalk(repository)) {
                RevCommit commit = rw.parseCommit(tagObject.getObjectId());
                treeId = commit.getTree().getId();
            }
        }
        
        if (treeId == null) return Optional.empty();
        
        try (TreeWalk tw = TreeWalk.forPath(repository, path, treeId)) {
            if (tw == null) return Optional.empty();
            byte[] bytes = repository.open(tw.getObjectId(0)).getBytes();
            String content = new String(bytes, StandardCharsets.UTF_8);
            return Optional.of(MetadataCodec.decode(content));
        }
    } catch (IOException e) {
        throw new StorageException("Failed to get artifact " + type + "/" + name, e);
    }
}
```

- Concern #1: **bundle() produces ZIP but design doc specifies tar.gz** — REST API endpoint `GET /artifacts/{type}/{name}/bundle?ref=…` is documented as returning tar.gz. Plan uses `ZipOutputStream`. These are incompatible formats. Either align with design doc (use TarArchiveOutputStream from commons-compress or similar) or update design doc to specify ZIP. Decision needed.

- Concern #2: **bundle() signature mismatch with design doc and REST API** — Same issue as flagged on TASK-0003.01. Plan takes `List<String> slugs` but design doc has `bundle(String type, String name, String ref)`. The REST API serves a single artifact at a specific version. The plan loses version resolution for bundles. Either keep design doc signature (which supports ref-based version lookup) or add a `ref` parameter.

- Concern #3: **bundle() with non-existent slug — behaviour undefined** — Test case 16 says "produces empty zip or throws" — this ambiguity must be resolved. Recommend: throw StorageException if any requested slug is not found (fail-fast, clear error for API callers).

Verdict: Plan needs revision — bundle format and signature must align with design doc. Non-existent slug behaviour must be specified concretely.
