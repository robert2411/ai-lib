## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
MetadataCodec parses YAML frontmatter (--- delimited) from raw artifact content into ArtifactMetadata. Serialises ArtifactMetadata back to YAML frontmatter. Handles missing/extra fields gracefully.
<!-- SECTION:DESCRIPTION:END -->

## Acceptance Criteria
<!-- AC:BEGIN -->
- [ ] #1 decode(encode(metadata)) returns equivalent metadata
- [ ] #2 Content body after frontmatter is preserved unchanged
- [ ] #3 Missing optional fields deserialise to null/empty without exception
- [ ] #4 agent-group frontmatter with members list round-trips correctly
<!-- AC:END -->

## Implementation Plan

<!-- SECTION:PLAN:BEGIN -->
1. Depends on: TASK-0002.01 (enums), TASK-0002.02 (ArtifactMetadata, Artifact records)
2. Create package directory: src/main/java/com/agentlibrary/metadata/
3. Create class MetadataCodec in com.agentlibrary.metadata.MetadataCodec:
   - File: src/main/java/com/agentlibrary/metadata/MetadataCodec.java
   - Public class with static methods (or Spring @Component — prefer static for testability)
   - Uses org.yaml.snakeyaml.Yaml for parsing/serialisation (already in pom.xml)
   - Method: public static Artifact decode(String rawContent)
     - Splits rawContent into frontmatter and body
     - Frontmatter delimited by lines starting with exactly "---"
     - First "---" opens frontmatter, second "---" closes it
     - Everything after second "---" is the body content
     - Parses YAML frontmatter into Map<String, Object> using SnakeYAML
     - Maps YAML fields to ArtifactMetadata fields:
       - "name" -> name (String)
       - "title" -> title (String)
       - "type" -> ArtifactType.fromSlug(value)
       - "version" -> version (String)
       - "description" -> description (String)
       - "harnesses" -> List<Harness> via Harness.fromSlug() for each
       - "tags" -> List<String>
       - "category" -> category, "language" -> language, "author" -> author
       - "visibility" -> Visibility.fromSlug(value)
       - "created"/"updated" -> Instant (parse ISO-8601)
       - "install" -> InstallConfig (map target, files, merge)
       - "members" -> List<AgentGroupMember> (each map with slug and role)
       - Unrecognised keys -> extra map
     - Missing optional fields default to null or empty collections (no exception)
     - Returns Artifact(metadata, bodyContent)
   - Method: public static String encode(Artifact artifact)
     - Serialises artifact.metadata() to a YAML map
     - Enum fields serialised as slug() value
     - Instant fields as ISO-8601 strings
     - Null fields omitted from YAML output
     - Empty lists/maps omitted from YAML output
     - Wraps in "---\n" + yaml + "---\n" + body
   - Method: public static String extractBody(String rawContent)
     - Returns everything after closing "---" frontmatter delimiter
   - Method: public static boolean hasFrontmatter(String rawContent)
     - Returns true if content starts with "---"
4. Create test directory: src/test/java/com/agentlibrary/metadata/
5. Create test file MetadataCodecTest:
   - File: src/test/java/com/agentlibrary/metadata/MetadataCodecTest.java
   - Test: decodeEncodeRoundTrip (AC #1)
   - Test: bodyPreservedUnchanged (AC #2)
   - Test: missingOptionalFields (AC #3)
   - Test: agentGroupMembersRoundTrip (AC #4)
   - Test: noFrontmatterThrows
   - Test: extraFieldsPreserved
   - Test: encodeOmitsNullFields
   - Test: encodeOmitsEmptyLists
   - Test: instantParsing (ISO-8601)
   - Test: harnessListParsing
   - Test: installConfigParsing
6. Implementation notes:
   - Use SnakeYAML DumperOptions with BLOCK flow style
   - Handle newlines: ensure single trailing newline after body
   - Use Representer if needed to avoid excessive quoting
