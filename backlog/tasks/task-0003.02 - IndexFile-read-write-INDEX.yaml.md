### Files to Create

#### 1. `src/main/java/com/agentlibrary/storage/IndexFile.java`

Utility class that reads/writes the INDEX.yaml file. INDEX.yaml is a flat list of artifact metadata entries maintained inside the git working tree (or bare repo object tree).

```java
package com.agentlibrary.storage;

import com.agentlibrary.model.ArtifactMetadata;
import java.util.List;

public class IndexFile {
    private IndexFile() {}

    public static List<ArtifactMetadata> load(String yamlContent);
    public static String save(List<ArtifactMetadata> entries);
}
```

**Design decisions:**
- `load(String yamlContent)` takes raw YAML string (not a file path). This keeps IndexFile decoupled from git — the caller (LocalGitArtifactRepository) reads the blob content from git and passes it in.
- `save(List<ArtifactMetadata>)` returns a YAML string. Caller writes it to git.
- Null or empty input to `load` returns `Collections.emptyList()` — satisfies AC#3.

**YAML structure of INDEX.yaml:**

```yaml
artifacts:
  - name: git-helper
    type: skill
    version: 1.2.0
    title: Git Helper
    description: One-line summary
    harnesses: [claude, copilot]
    tags: [git, workflow]
    category: developer-tools
    author: robert
    visibility: team
    created: 2026-04-19T10:00:00Z
    updated: 2026-04-19T10:00:00Z
  - name: other-artifact
    ...
```

**Implementation steps:**

1. **`load(String yamlContent)`:**
   - If `yamlContent` is null or blank, return `List.of()` (AC#3)
   - Parse YAML using SnakeYAML `new Yaml().load(yamlContent)` → `Map<String, Object>`
   - Extract `artifacts` key → `List<Map<String, Object>>`
   - For each map entry, convert to `ArtifactMetadata` using same field-mapping logic as MetadataCodec.mapToMetadata (extract into shared utility or duplicate minimal mapping)
   - Return unmodifiable list
   - If `artifacts` key missing or not a list, return `List.of()`

2. **`save(List<ArtifactMetadata> entries)`:**
   - Create `Map<String, Object>` with key `"artifacts"` → list of maps
   - For each ArtifactMetadata, convert to Map (slug-based enum values, ISO-8601 timestamps)
   - Serialize with SnakeYAML DumperOptions: BLOCK flow style, explicit start (---)
   - Return the YAML string

3. **Shared mapping concern:**
   - MetadataCodec already has `mapToMetadata` and `metadataToMap` (private). 
   - **Approach:** Extract a package-private `MetadataMapper` helper in `com.agentlibrary.metadata` with static methods `toMap(ArtifactMetadata)` and `fromMap(Map<String,Object>)`. 
   - Alternatively, make MetadataCodec methods package-private or add public static utility methods.
   - **Recommended:** Add two public static methods to MetadataCodec:
     - `MetadataCodec.metadataFromMap(Map<String,Object>): ArtifactMetadata`
     - `MetadataCodec.metadataToMap(ArtifactMetadata): Map<String,Object>`
   - This avoids code duplication between IndexFile and MetadataCodec.
   - The Implementation agent should refactor MetadataCodec to expose these two methods as public static.
