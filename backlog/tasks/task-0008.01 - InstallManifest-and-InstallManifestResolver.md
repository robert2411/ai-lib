## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
InstallManifest record: list of InstallEntry (source path in repo, targetPath resolved per harness). InstallManifestResolver resolves target paths for each artifact type + harness combination using a harness path config map. For agent-group slugs: resolver expands the group into its member agents and returns one InstallEntry per member, preserving role order (manager, analyse, implementation).
<!-- SECTION:DESCRIPTION:END -->
