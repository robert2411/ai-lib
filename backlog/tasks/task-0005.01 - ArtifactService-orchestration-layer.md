## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
ArtifactService coordinates MetadataCodec, MetadataValidator, ArtifactRepository, IndexService, SearchService. Methods: createOrUpdate, get, list, search, delete, getVersions, bundle. For agent-group artifacts: resolveGroup(slug) expands the group into its ordered member agents (manager first, then analyse, then implementation) and returns them as a list. Validates all member slugs exist before saving a group.
<!-- SECTION:DESCRIPTION:END -->

## Acceptance Criteria

## Implementation Plan

<!-- SECTION:PLAN:BEGIN -->
1. Create package com.agentlibrary.service

2. Create NotFoundException (src/main/java/com/agentlibrary/service/NotFoundException.java)
   - Extends RuntimeException
   - Constructor takes String message
   - Used when slug/version not found in repo

3. Create ArtifactService class (src/main/java/com/agentlibrary/service/ArtifactService.java)
   - @Service annotation
   - Constructor-injected dependencies:
     * MetadataCodec (static methods, no injection needed)
     * ArtifactRepository repository
     * IndexService indexService
     * SearchService searchService
   - Note: MetadataValidator is static, called directly

4. Implement createOrUpdate(String rawContent, User user) -> CommitResult
   - Call MetadataCodec.decode(rawContent) to get Artifact
   - Call MetadataValidator.validate(artifact.metadata()) — throws ValidationException on failure (AC #1)
   - If type is AGENT_GROUP, call validateGroupMembers(artifact.metadata()) — throws ValidationException if any member slug not found (AC #6)
   - Build commit message: "<type>/<name>: create/update by <user>"
   - Call repository.save(artifact, message, user)
   - Return CommitResult

5. Implement get(String slug, String ref) -> Artifact
   - Look up artifact type from IndexService cache by matching slug to metadata.name()
   - Call repository.get(type, slug, ref)
   - Throw NotFoundException if Optional is empty

6. Implement list(Filter filter) -> List<ArtifactMetadata>
   - Delegate to indexService.getAll() then filter in-memory by Filter fields (AC #3)
   - Or delegate to repository.list(filter) if filter has no text query
   - Apply pagination (page, size) — return subList

7. Implement search(String query, Filter filter) -> List<ArtifactMetadata>
   - Delegate to searchService.search(query, filter) (AC #2)
   - Return results

8. Implement delete(String slug, String ref, User user) -> void
   - Resolve type from slug via index
   - Call repository.delete(type, slug, commitMessage, user)
   - Throw NotFoundException if slug unknown

9. Implement getVersions(String slug) -> List<VersionRef>
   - Resolve type from slug via index
   - Call repository.versions(type, slug)

10. Implement bundle(String slug, String ref) -> InputStream
    - Resolve type from slug via index
    - Call repository.bundle(type, slug, ref)

11. Implement resolveGroup(String slug) -> List<ResolvedGroupMember>
    - Get artifact by slug (latest)
    - Verify type == AGENT_GROUP, else throw IllegalArgumentException
    - For each AgentGroupMember in metadata.members():
      * Look up the member artifact by member.slug() (latest)
      * Build ResolvedGroupMember(slug, role, metadata)
    - Sort by role order: manager=0, analyse=1, implementation=2, custom=3 (AC #5)
    - Return sorted list

12. Create ResolvedGroupMember record (src/main/java/com/agentlibrary/service/ResolvedGroupMember.java)
    - Fields: String slug, String role, ArtifactMetadata metadata

13. Private helper: resolveType(String slug) -> ArtifactType
    - Search indexService.getAll() for entry where name == slug
    - Throw NotFoundException if not found

14. Private helper: validateGroupMembers(ArtifactMetadata metadata)
    - For each member in metadata.members(), check resolveType(member.slug()) exists
    - Collect missing slugs, throw ValidationException with list of errors (AC #6)

15. Unit tests (src/test/java/com/agentlibrary/service/ArtifactServiceTest.java)
    - Mock ArtifactRepository, IndexService, SearchService (AC #4)
    - Test createOrUpdate happy path — verify validator called, repo.save called
    - Test createOrUpdate with invalid metadata — expect ValidationException
    - Test createOrUpdate agent-group with non-existent member — expect ValidationException (AC #6)
    - Test get happy path — returns artifact
    - Test get unknown slug — throws NotFoundException
    - Test search delegates to SearchService with filter (AC #2)
    - Test list delegates to IndexService with filter (AC #3)
    - Test resolveGroup returns members sorted: manager, analyse, implementation, custom (AC #5)
    - Test resolveGroup for non-group artifact — throws IllegalArgumentException
    - Test delete/getVersions/bundle delegation

## Implementation Notes

- resolveType() helper scans IndexService cache — acceptable for <10k artifacts.
- IllegalArgumentException from MetadataCodec.decode() (no frontmatter) will propagate to ApiExceptionHandler as 400.
- list() filters in-memory from indexService.getAll() for non-text-query filters; search() delegates to SearchService for text queries.
Analysis complete. Plan ready. No blockers.
