## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
ApiExceptionHandler: @RestControllerAdvice mapping ValidationException to 422, NotFoundException to 404, generic to 500 with RFC 7807 ProblemDetail. LookupApiController: GET /api/v1/tags, /types, /harnesses. AdminApiController: POST /api/v1/admin/reindex.
<!-- SECTION:DESCRIPTION:END -->

## Acceptance Criteria
<!-- AC:BEGIN -->
- [ ] #1 ValidationException returns 422 with field errors in body
- [ ] #2 GET /api/v1/tags returns distinct tag list
- [ ] #3 POST /api/v1/admin/reindex triggers index rebuild, requires ROLE_ADMIN
<!-- AC:END -->

## Implementation Plan

<!-- SECTION:PLAN:BEGIN -->
1. Create ApiExceptionHandler (src/main/java/com/agentlibrary/web/api/ApiExceptionHandler.java)
   - @RestControllerAdvice(basePackages = "com.agentlibrary.web.api")
   - Uses RFC 7807 ProblemDetail (Spring 6 built-in support via org.springframework.http.ProblemDetail)

2. Handle ValidationException → 422 (AC #1)
   - @ExceptionHandler(ValidationException.class)
   - Create ProblemDetail with status 422 (UNPROCESSABLE_ENTITY)
   - Set title: "Validation Failed"
   - Set detail: exception.getMessage()
   - Set property "errors": exception.errors() — list of field-level error strings (AC #1)
   - Return ResponseEntity<ProblemDetail>

3. Handle NotFoundException → 404
   - @ExceptionHandler(NotFoundException.class)
   - ProblemDetail with status 404
   - Set title: "Not Found"
   - Set detail: exception.getMessage()

4. Handle IllegalArgumentException → 400
   - @ExceptionHandler(IllegalArgumentException.class)
   - ProblemDetail with status 400
   - Set title: "Bad Request"
   - Set detail: exception.getMessage()
   - Used for: non-group artifact on /members endpoint, slug mismatch on PUT

5. Handle StorageException → 500
   - @ExceptionHandler(StorageException.class)
   - ProblemDetail with status 500
   - Set title: "Storage Error"
   - Set detail: "An internal storage error occurred" (don't leak internals)

6. Handle generic Exception → 500
   - @ExceptionHandler(Exception.class)
   - ProblemDetail with status 500
   - Set title: "Internal Server Error"
   - Set detail: "An unexpected error occurred"
   - Log the actual exception at ERROR level

7. Create LookupApiController (src/main/java/com/agentlibrary/web/api/LookupApiController.java)
   - @RestController @RequestMapping("/api/v1")
   - Inject FacetService

8. Implement GET /api/v1/tags (AC #2)
   - Method: getTags() → List<String>
   - Call facetService.getDistinctTags()
   - Return 200 with JSON array of tag strings

9. Implement GET /api/v1/types
   - Method: getTypes() → List<String>
   - Call facetService.getDistinctTypes()
   - Map ArtifactType → slug strings
   - Return 200 with JSON array

10. Implement GET /api/v1/harnesses
    - Method: getHarnesses() → List<String>
    - Call facetService.getDistinctHarnesses()
    - Map Harness → slug strings
    - Return 200 with JSON array

11. Create AdminApiController (src/main/java/com/agentlibrary/web/api/AdminApiController.java)
    - @RestController @RequestMapping("/api/v1/admin")
    - Inject IndexService

12. Implement POST /api/v1/admin/reindex (AC #3)
    - Method: reindex()
    - @PreAuthorize("hasRole('ADMIN')") — or rely on SecurityConfig URL rules
    - Call indexService.refresh() to trigger full index rebuild from repository
    - Return 200 with body: {"status": "ok", "message": "Index rebuilt successfully"}
    - SecurityConfig ensures only ADMIN can access /api/v1/admin/** (AC #3)

13. Unit tests for ApiExceptionHandler (src/test/java/com/agentlibrary/web/api/ApiExceptionHandlerTest.java)
    - @WebMvcTest with a test controller that throws each exception type
    - Test ValidationException → 422 with "errors" field in JSON body (AC #1)
    - Test NotFoundException → 404 with RFC 7807 format
    - Test IllegalArgumentException → 400
    - Test generic Exception → 500
    - Verify Content-Type: application/problem+json

14. Unit tests for LookupApiController (src/test/java/com/agentlibrary/web/api/LookupApiControllerTest.java)
    - @WebMvcTest(LookupApiController.class)
    - @MockBean FacetService
    - @WithMockUser
    - Test GET /api/v1/tags returns JSON array of strings (AC #2)
    - Test GET /api/v1/types returns type slugs
    - Test GET /api/v1/harnesses returns harness slugs

15. Unit tests for AdminApiController (src/test/java/com/agentlibrary/web/api/AdminApiControllerTest.java)
    - @WebMvcTest(AdminApiController.class)
    - @MockBean IndexService
    - Test POST /api/v1/admin/reindex with ADMIN role → 200 (AC #3)
    - Test POST /api/v1/admin/reindex with USER role → 403 (AC #3)
    - Test POST /api/v1/admin/reindex unauthenticated → 401
    - Verify indexService.refresh() is called exactly once
