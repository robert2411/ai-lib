## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
LoginRateLimiter intercepts POST /login, applies per-IP bucket4j token bucket (e.g. 10 attempts / 10 minutes). Exceeded limit returns 429 Too Many Requests. Integrated via Spring Security filter chain.
